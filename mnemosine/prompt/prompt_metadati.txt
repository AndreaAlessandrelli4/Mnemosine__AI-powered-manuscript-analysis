REGOLE GENERALI (OBBLIGATORIE)
- Usa SOLO informazioni direttamente visibili nelle immagini (testo, decorazioni, note, sigilli, stato materiale, ecc.).
- NON usare conoscenze esterne, memoria addestrata o inferenze (niente “probabilmente”, niente periodi/luoghi/autori non espliciti).
- NON riempire per intuizione: se non sei sicuro, lascia stringa vuota: "".
- Non usare: null, true/false, liste/array, numeri (a meno che facciano parte del testo trascritto nei campi testuali).
- Non scrivere commenti o spiegazioni. Se qualcosa è illeggibile o incerto → "".
- L’output deve essere JSON valido: doppie virgolette, niente trailing comma.

ISTRUZIONI PER I CAMPI

1) NOTAZIONE_MUSICALE
- presenza: "SÌ" se vedi chiaramente righi/segni musicali; "NO" se non vedi notazione; "" se incerto.
- tipo: SOLO se presenza="SÌ". Valori ammessi (MAIUSCOLO, separati da ", " se multipli):
  MODERNA, LETTERALE, NEUMATICA, INTAVOLATURA, QUADRATA, ALFABETO
  Se non determinabile → "".
- colore_note: SOLO se presenza="SÌ". Indica colore/i (max 30 caratteri), es: "nero", "rosso", "nero e rosso". Se incerto → "".

2) PALINSESTO
- presenza: "SÌ" solo se vedi chiaramente scrittura sottostante cancellata/sovrapposta; "NO" se non vedi tracce; "" se incerto.
- tipo: SOLO se presenza="SÌ". Valori ammessi: "INTEGRALE" oppure "PARZIALE". Se incerto → "".

3) DECORAZIONI
Tutti i sotto-campi: SOLO "SÌ" / "NO" / "".
- iniziali_semplici: iniziali ingrandite/evidenziate senza decorazione complessa.
- iniziali_filigranate: iniziali con filigrana/svolazzi a penna.
- iniziali_ornate: motivi decorativi (foglie/viticci/geometrie/oro/colori) senza figure narrative.
- iniziali_istoriate: figure umane/animali/scene riconoscibili dentro l’iniziale.
- pagine_ornate: decorazioni non figurative significative (cornici, fregi, motivi) senza grandi scene autonome.
- pagine_illustrate: miniature/illustrazioni figurative autonome (scene/personaggi/grandi immagini).

4) STATO_DI_CONSERVAZIONE
Scegli UNA tra: "PESSIMO", "CATTIVO", "DISCRETO", "BUONO", "OTTIMO".
Se non determinabile → "".

5) DESCRIZIONE_GENERALE
Breve descrizione sintetica (max 2–4 frasi), SOLO su ciò che vedi (supporto, scrittura, layout, decorazioni, danni, elementi notevoli).
Niente ipotesi o contesto esterno. Se non puoi → "".

6) NOMI
Trascrivi NOMI propri di persone visibili (come appaiono). Solo nomi, separati da virgole.
Non aggiungere ruoli (niente “autore/copista”). Se nessuno → "".

7) TITOLO
Trascrivi fedelmente il titolo se chiaramente presente (frontespizio/copertina/incipit evidente). Mantieni grafia/maiuscole/abbreviazioni.
Se non chiaramente identificabile → "".

8) DATAZIONE
Compila SOLO se è presente una formula di data esplicita e riconosci il sistema di datazione.
Valori ammessi (esatti):
"ERA_CRISTIANA"
"ERA_BIZANTINA"
"ERA_DELLA_FONDAZIONE_DI_ROMA"
"ERA_DI_DIOCLEZIANO_O_DEI_MARTIRI"
"EGIRA"
Altrimenti → "".

9) PRESENZA_SIGILLI_TIMBRI
"SÌ" se vedi chiaramente sigilli/timbri (istituzioni/collezioni), "NO" se assenti, "" se incerto.
Non considerare semplici firme/annotazioni manoscritte come timbri.

10) STRUTTURA_MATERIALE
Scegli UNA:
"FASCICOLI LEGATI" (volume rilegato)
"FRAMMENTO" (singolo foglio o poche carte sciolte)
Se incerto → "".

11) LINGUA
Indica la lingua principale (es. "Italiano", "Latino") basandoti SOLO su parole leggibili.
Se due lingue principali chiarissime: "Latino, Italiano". Se incerto → "".

12) TIPO_PAGINA
Scegli UNA etichetta:
"PAGINA_INTERNA" | "COPERTINA" | "FRONTESPIZIO" | "RILEGATURA" | "DORSO" | "CARTE_DI_GUARDIA" | "TAGLIO" | "ALTRO"
Se questa chiamata contiene più immagini di tipi diversi, scegli il tipo prevalente; se non prevale chiaramente → "ALTRO".
Se incerto → "".

13) NOTE_ESTERNE_AL_TESTO
"SÌ" se vedi note marginali/interlineari/postille/glosse aggiunte (non rubriche/titoli strutturali del testo base).
"NO" se assenti. "" se incerto.

14) IMPAGINAZIONE
Scegli UNA:
"COLONNA_SINGOLA" | "DUE_COLONNE" | "TRE_COLONNE_O_PIÙ" | "LAYOUT_MISTO"
Se in più immagini vedi chiaramente layout diversi → "LAYOUT_MISTO".
Se non determinabile (poche righe, solo miniature, dettagli) → "".

OUTPUT (OBBLIGATORIO)
Restituisci SOLO il JSON valido, con la struttura esatta e valori secondo le regole sopra. Nessun testo extra.
